A backup bundles your config and all runtime state under var/ — the SQLite item history (rssrob.db), generated feeds, the email subscriber list, digest state, and the 公众号 credential — into one .zip. Use it to move to another machine or rebuild after a reinstall.

Download a backup

Saves the current config + var/ as a timestamped zip.

⚠️ The zip contains secrets (your 公众号 cookie/token and subscriber emails). Keep it private.

Restore from a backup

Extracts the zip back into place, overwriting the current config + var/.

After restoring, restart rssrob serve — the running scheduler holds the old database connection until then.
