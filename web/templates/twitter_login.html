RSSRob reads X with your own logged-in session. It calls the same web API the x.com site uses, sending your auth_token + ct0 cookies. Nothing is posted; the cookie stays local (var/twitter_credential.json, gitignored). It expires after a while — just paste a fresh one.

After logging in to x.com, grab your cookie:

  1. DevTools (F12) → Network → click any request to x.com → Headers → Request Headers → copy the whole Cookie value.
  2. ⚠️ Don't use document.cookie — auth_token is HttpOnly and won't appear there.
